# Google Ads — импорт через Editor

## Файлы

- `myavto-keywords.csv` — 150+ ключевых слов в точном и фразовом соответствии, разбитые по 30+ группам объявлений с привязкой к посадочным страницам.
- `myavto-negatives.csv` — 100+ минус-слов уровня кампании (марки не-японцев, мусорные запросы).

## Как импортировать

1. Открой **Google Ads Editor** (десктопное приложение).
2. Account → выбери `myavto`.
3. Account → Import → From file → выбери `myavto-keywords.csv`.
4. Просмотри изменения в Preview, нажми Apply.
5. Повтори для `myavto-negatives.csv` (Import → From file).
6. Post changes → отправит в облако.

## Структура

Все ключи в одной кампании `MY AVTO Search` (используется существующая, не создаёт новую — Editor смержит).

Группы объявлений:
- Поршни общее / Кольца общее / Вкладыши общее
- Toyota / Nissan / Mitsubishi / Mazda
- 30+ групп по двигателям (1KZ, 2JZ, VQ35, 4G63 и т.д.)
- 6 групп по брендам запчастей (TEIKIN, NPR, IZUMI, RIK, TP, ND)
- Гео-Алматы

## Важно перед запуском

1. Посадочные страницы `/zapchasti/vkladyshi/` и `/brendy/{brand}/` пока **не существуют** на сайте — клики туда будут давать 404. Либо создать эти страницы (см. roadmap), либо временно перенаправить эти группы на `/zapchasti/porshni/` или `/`.
2. Бюджет $8/день — стартуй с 3 групп: Поршни общее, Кольца общее, Nissan-двигатели. Остальные сначала на паузу.
3. Добавь объявления вручную через интерфейс или Editor — CSV содержит только ключи. Шаблон заголовка:
   - H1: `Поршни {ENGINE} в Алматы`
   - H2: `TEIKIN, NPR, IZUMI · В наличии`
   - Description: `Кольца, поршни, вкладыши для капремонта. Доставка по РК. Гарантия. Kaspi рассрочка.`
